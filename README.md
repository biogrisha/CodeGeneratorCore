# Code Generation Pipeline

## Этапы генерации кода

Процесс генерации кода состоит из следующих этапов:

1) Парсинг xml в промежуточное представление. 
2) Формирование e-графа программы из термов 
3) Применение e-matching реализованный по алгоритму из https://leodemoura.github.io/files/ematching.pdf 
(Я использовал свой код, над которым работаю около месяца для своего проекта системы комп. алгебры) 

4) Применение сатурации и упрощения выражений по аналогии как это делает egg 
5) Составление конечного документа
